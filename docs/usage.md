## Usage

You define the grid size and RobIT’s initial position. Then issue commands:

- `L` — Turn left
- `R` — Turn right
- `F` — Move forward
- `B` — Move backward

RobIT exits gracefully if it tries to go beyond the grid.

### Example

```text
    Enter grid width: 5
    Enter grid height: 5
    Enter RobIT's x: 2
    Enter RobIT's y: 3
    Enter direction: N
    Enter commands: FFRLF
```
